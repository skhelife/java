# 🎓 学生成绩管理系统 v2.0 Enhanced Edition

## 🌟 项目概述
这是一个**极具视觉冲击力**的基于命令行的学生成绩管理系统，使用Java语言开发。系统完全采用内存存储，不使用任何数据库，所有数据在程序启动时动态生成。

### ✨ 新版本亮点
- 🎨 **炫酷的ASCII艺术字界面**
- 🌈 **丰富的颜色和特效**
- 📊 **美观的表格显示**
- 🎯 **直观的进度条和图表**
- 🏆 **互动性强的用户体验**

## 📁 项目结构
```
src/
└── com/
    └── yourname/
        ├── model/              # 实体类
        │   ├── Student.java        # 学生实体
        │   ├── Teacher.java        # 教师实体
        │   ├── Course.java         # 课程实体
        │   ├── TeachingClass.java  # 教学班实体
        │   └── Grade.java          # 成绩实体
        ├── service/            # 业务逻辑层
        │   └── EducationService.java  # 教育服务类（单例模式）
        ├── ui/                 # 用户界面层
        │   ├── UIUtils.java        # UI工具类（新增）
        │   └── CommandLineUI.java     # 命令行界面
        └── Main.java           # 程序入口
```

## 🚀 系统功能
1. **📊 查看教学班成绩** - 美观表格显示，支持按学号或综合成绩排序
2. **📈 成绩分布统计** - 可视化图表展示各分数段分布
3. **🔍 查询学生成绩** - 个性化成绩单，包含排名信息
4. **🏆 学生总成绩排名** - 荣誉榜展示，前三名特别标识
5. **📋 教学班信息总览** - 结构化展示所有教学班信息
6. **🎯 系统统计信息** - 全面的数据分析和统计（新增）

## 📊 数据规模
- 👨‍🎓 学生：**300人**（大幅增加）
- 👩‍🏫 教师：10人
- 📚 课程：5门（Java程序设计、数据结构与算法、计算机网络、数据库原理、操作系统）
- 🏛️ 教学班：10个（每门课程2个班）
- 📋 成绩记录：根据学生选课情况动态生成（约1200+条记录）

## 运行方式

### 方式一：使用批处理文件
双击 `run.bat` 文件即可自动编译并运行程序。

### 方式二：手动编译运行
1. 编译项目：
```bash
javac -d . src/com/yourname/model/*.java src/com/yourname/service/*.java src/com/yourname/ui/*.java src/com/yourname/*.java
```

2. 运行程序：
```bash
java com.yourname.Main
```

## 💡 技术特点
1. **🔧 单例模式** - EducationService采用单例模式确保全局唯一的数据源
2. **⚡ Lambda表达式** - 在排序功能中使用Lambda表达式实现Comparator
3. **🛡️ 泛型应用** - 大量使用Java泛型提高代码安全性
4. **🌊 Stream API** - 使用Java 8 Stream API进行数据过滤和处理
5. **🎨 UI工具类** - 专门的UIUtils类提供丰富的界面美化功能（新增）
6. **🎯 面向对象设计** - 充分应用封装、继承等面向对象特性
7. **📊 数据可视化** - 文本模式下的图表和进度条展示

## 成绩计算规则
综合成绩 = 平时成绩 × 20% + 期中成绩 × 20% + 实验成绩 × 30% + 期末成绩 × 30%

## 数据生成规则
- 每个学生至少选择3门课程
- 每个教学班至少有20个学生
- 所有成绩都在60-100分范围内随机生成
- 每门课程至少有2位老师授课（2个教学班）
